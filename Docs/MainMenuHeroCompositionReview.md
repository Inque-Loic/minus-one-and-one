# 主界面 Hero 构图施工评审与调整建议

更新时间：2026-06-11  
评审对象：按 `Docs/MainMenuHeroCompositionDesign.md` 施工后的主界面截图

## 1. 总体结论

当前版本方向是对的：标题和菜单已经从中心移到左侧，右侧加入了身份卡、棋子、筹码，整体比早期中央按钮版更接近“标题页插画”。

但距离参考图的封面感还差两点：

1. **右侧主体还不够突出**：卡牌和棋子偏小、偏低、偏像装饰，还没有成为玩家第一眼看到的主视觉。
2. **左侧承托仍然太像大面板**：半透明矩形边界明显，视觉上仍像 UI 框，而不是自然融进背景的标题区。

下一轮建议不要继续加新元素，而是做“减面板、放主体、强标题、弱按钮框”。

## 2. 当前优点

| 项目 | 评价 |
| --- | --- |
| 构图方向 | 已从中央菜单改成左侧信息区 + 右侧主体区，方向正确。 |
| 背景保留 | 仙山、宫殿、云海没有被完全遮挡，画面空间还在。 |
| 桌游元素 | 右侧卡牌、棋子、筹码已经建立“身份推理”提示。 |
| 主按钮 | “开始游戏”权重明显，颜色和可点击感较强。 |
| 副标题 | “云上疑局 / 接触、声明与猜分”比长句更适合标题页。 |

## 3. 主要问题

### 3.1 左侧半透明矩形太硬

当前左侧有一个大面积半透明矩形，右边界清楚，覆盖到画面中部。它解决了可读性，但带来两个问题：

- 看起来像调试用面板或占位框。
- 把背景分割成左右两块，削弱了标题页插画感。

建议：

- 不要使用硬边矩形。
- 改成从左向右逐渐消失的暗色渐变。
- 渐变覆盖左侧 35%-42% 宽度即可，不要进入画面中心。

推荐效果：

```text
左侧深色 45%  →  中部透明 0%
```

### 3.2 标题还不够像 Logo

标题位置已经左移，但大小和质感仍偏普通标题文字。参考图里的标题/Logo 通常非常强，承担品牌识别。

建议：

- 标题字号再放大 20%-30%。
- 增加更明显的深色描边或阴影。
- 标题整体向左移动一点，靠近左侧视觉轴。
- 标题和副标题之间拉开层级，副标题不要抢标题。

### 3.3 右侧主体太小、太散

右侧卡牌、棋子、筹码目前位置偏低，尺寸偏小，更像 UI 装饰，不像主体。参考图的主体往往占画面 35%-55% 的视觉面积。

建议：

- 把两张身份卡放大到当前的 1.8-2.4 倍。
- 卡牌位置上移到右侧中部云海区域，不要只压在右下角。
- 绿色卡和红色卡做交叉或前后错位，形成对峙。
- 棋子放在卡牌前景，作为阵营对抗点。
- 给卡牌/棋子加轻微阴影或光晕，避免漂浮得像 UI 图标。

推荐主体位置：

| 元素 | 建议位置 |
| --- | --- |
| 绿卡 | 屏幕 X 70%-74%，Y 48%-58%，略向左倾斜 |
| 红卡 | 屏幕 X 76%-80%，Y 54%-65%，略向右倾斜 |
| 绿棋子 | 卡牌前方左下 |
| 红棋子 | 卡牌前方右下 |
| 筹码 | 棋子底部少量点缀 |

### 3.4 菜单仍然偏按钮面板感

参考图里的菜单多是文字列表，按钮框很弱。当前两个矩形按钮仍有功能面板感，尤其“游戏测试”像灰色禁用按钮。

建议：

- 如果要更接近参考图，按钮可改为轻文字列表。
- 主项“开始游戏”保留金色高亮和箭头。
- 次项“游戏测试”用白色/青白文字，不要像不可用状态。
- 如果保留按钮底图，按钮宽度缩短 15%-20%，透明度降低，避免抢标题。

推荐菜单：

```text
▶ 开始游戏
  游戏测试
```

文字需要有点击热区，但视觉上可以不像大按钮。

### 3.5 右侧宫殿主体被雾化得过弱

背景最有价值的主体是右侧宫殿和日光，但现在整体偏白、偏雾，细节不够跳。

建议：

- 不要继续全屏加白雾或高亮。
- 对右侧宫殿区域做轻微对比度提升。
- 左侧暗化，右侧主体自然会更突出。
- 保留日光，但避免让整个右上泛白吃掉建筑轮廓。

## 4. 下一轮优先级

### P0：必须调整

| 任务 | 目标 |
| --- | --- |
| 左侧硬矩形改渐变 | 消除 UI 面板感。 |
| 右侧卡牌/棋子放大并上移 | 让主体真正成立。 |
| 标题放大并加阴影/描边 | 提高品牌识别。 |
| 副按钮提升可读性 | 避免“游戏测试”看起来不可用。 |

### P1：建议调整

| 任务 | 目标 |
| --- | --- |
| 菜单改成文字列表风格 | 更贴近参考图。 |
| 右侧宫殿增强对比 | 主体更有存在感。 |
| 左下棋盘略微提亮 | 强化桌游锚点。 |
| 加小版本号 | 增加发布页完整度。 |

### P2：可选

| 任务 | 目标 |
| --- | --- |
| 标题后加淡金圆阵 | 增加仪式感，但透明度要低。 |
| 卡牌周围加轻微粒子/光点 | 提升氛围，不影响可读。 |

```text
宫殿和日光保留为远景主体
两张放大的红绿身份卡在右中错位漂浮

筹码少量点缀
```

## 6. 对执行会话的具体修改指令

```text
请基于当前主菜单效果继续微调，不要重新推翻。

优先做四件事：
1. 将左侧半透明硬矩形替换为左到右渐隐的暗色渐变，避免明显面板边界。
2. 将右侧红绿卡牌和棋子主体放大 1.8-2.4 倍并上移到右侧中部，让它们成为主视觉，而不是角落装饰。
3. 将“负一和一”标题放大 20%-30%，增加深色描边/阴影，使其更像 Logo。
4. 调整“游戏测试”视觉，让它看起来是可点击的次级入口，而不是禁用按钮。

如果还有时间，再把两个按钮改成更轻的文字菜单风格：
▶ 开始游戏
  游戏测试

完成后检查：
- 3 秒内能看清游戏名。
- 第一眼能看到右侧卡牌/棋子主体。
- 左侧没有明显大矩形面板感。
- 开始游戏和游戏测试都明显可点击。
- 1280x720 与 1920x1080 不重叠、不挡点击。
```

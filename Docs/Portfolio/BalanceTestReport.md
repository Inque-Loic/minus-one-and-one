## 3. 当前判断

当前版本偏向一阵营，但没有达到阻塞发布的程度。1000 局测试未卡死，说明回合推进、AI 行动和结算流程具备基本稳定性。

该数据可以作为 v0.1.1 的公开 Demo 基线。后续如果继续调平衡，应优先观察以下方向：

- 负一胜率是否需要提升到更接近 40%-45%。
- 一阵营平均分是否长期高于负一。
- AI 接受率 72.1% 是否让接触过于频繁。
- 猜分参与率 80.3% 是否过高，是否需要更明显的风险。

## 4. 可尝试的调参方向

| 方向 | 预期影响 |
| --- | --- |
| 提高负一行动收益 | 增加负一胜率。 |
| 降低一阵营安全接触收益 | 减少一阵营稳定滚分。 |
| 提高错误猜分成本 | 让猜分更谨慎。 |
| 调整 AI 拒绝倾向 | 控制接触频率。 |

## 5. 结论

v0.1.1 可以作为可发布 Demo。下一次平衡改动后，应重新记录 1000 局测试，避免只凭单局体验判断胜率。

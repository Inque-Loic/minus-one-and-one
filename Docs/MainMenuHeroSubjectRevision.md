# 主界面右侧主体修正方案

更新时间：2026-06-11  
背景：当前右侧使用 Kenney 卡牌、棋子、筹码作为主体后，视觉显得简陋，与仙山云海背景的绘制精度不匹配。

## 1. 结论

右侧 **不建议继续用 Kenney 卡牌、棋子、筹码作为主视觉主体**。

这些资源适合：

- 按钮图标
- 小型状态符号
- 猜分页装饰
- 玩家列表头像
- 讨论消息类型提示

但不适合作为主菜单 Hero 主体，因为它们是扁平 UI 素材，而当前背景是高完成度插画。两者放在同一视觉层级时，会产生“贴图感”和“临时素材感”。

## 2. 新主体方向

主界面右侧主体应从“素材拼贴”改为“背景内生主体”。

推荐方向：

> 以右侧仙山宫殿为主体，辅以轻量、半透明、绘制感更强的身份/棋局符号。

也就是说，主体不是“几张贴上去的卡牌和棋子”，而是：

- 宫殿
- 云路
- 日光
- 淡金阵法
- 半透明牌影
- 红绿两道细光

让玩家感觉这些符号属于这张背景，而不是 UI 元件贴在背景上。

## 3. 立即修改建议

### 3.1 删除或弱化当前右侧素材

| 元素 | 建议 |
| --- | --- |
| 红绿卡牌 | 删除，或缩小并降到 10%-18% 透明度，作为远景牌影。 |
| 棋子 | 删除，不再作为主菜单主体。 |
| 筹码 | 删除，不放在主菜单右侧。 |

原因：

- 当前尺寸和样式过于 UI 化。
- 与背景透视、光照、笔触不一致。
- 破坏右下塔阁和云海本身的精致感。

### 3.2 强化现有背景主体

把右侧主体改成“宫殿 + 云路 + 日光”：

| 区域 | 处理 |
| --- | --- |
| 右侧宫殿 | 略微提高对比度和清晰度，成为主视觉。 |
| 云中石路 | 轻微提亮，引导视线从左下棋盘走向右侧宫殿。 |
| 日光区域 | 保留亮度，但避免过曝。 |
| 右下塔阁 | 保持清晰，不被贴片遮挡。 |

### 3.3 用更克制的推理符号

替代卡牌/棋子/筹码：

| 替代元素 | 视觉方式 | 位置 |
| --- | --- | --- |
| 淡金阵法圆环 | 线稿、低透明度 8%-14% | 宫殿后方或右上云层 |
| 两张牌影 | 只保留轮廓，不用实体卡牌图 | 宫殿前景云雾中 |
| 红绿光点 | 小面积、柔和光晕 | 云路两侧或牌影边缘 |
| 细线连接 | 像推理线索，不要太亮 | 左下棋盘指向右侧宫殿 |

这些元素更像“画面中的神秘符号”，不会抢背景质感。

## 4. 两种可选主体方案

### 方案 A：仙山宫殿主体，推荐

画面重点：

- 右侧宫殿作为唯一大主体。
- 宫殿后有淡金阵法圆环。
- 云路上有非常淡的红绿线索光点。
- 左侧标题菜单和右侧宫殿形成对角线。

优点：

- 成本低。
- 不需要新画角色。
- 最贴合当前背景。
- 画面高级感最好。

缺点：

- 游戏的“阵营对抗”表达会更隐晦，需要靠标题、副标题和菜单图标补足。

### 方案 B：云中双牌主体

画面重点：

- 右侧云海中出现两张大型半透明“身份牌影”。
- 牌影不使用 Kenney 卡牌实体图，而是用简洁矩形轮廓、金色描边、轻微红绿光区分。
- 牌内可以只放抽象符号：`+1` 与 `-1`，或“一”和“负一”的篆印感符号。

优点：

- 主题更明确。
- 比 Kenney 卡牌更统一。
- 适合后续做成正式 Logo/海报符号。

缺点：

- 需要 UI 实现自制牌影，或生成/绘制更统一的牌面。

### 方案 C：角色立绘主体，暂不推荐

画面重点：

- 右侧加入一名原创角色立绘。

不推荐原因：

- 当前项目没有角色系统，强行加人物会改变产品气质。
- 需要新美术资产，成本高。
- 角色和背景风格难统一。

## 5. 推荐最终方案

采用 **方案 A + 少量方案 B 元素**：

```text
左侧：
标题、短副标题、文字菜单
左侧暗色渐变承托

右侧：
仙山宫殿作为主体
宫殿后方一圈淡金阵法
云海中两张几乎透明的身份牌影
牌影边缘只有很轻的红/绿光
不再使用实体棋子和筹码
```

这样既有主体，也不会让右侧变成廉价素材拼贴。

## 6. 具体修改指令

```text
请修正当前主菜单右侧主体。

当前右侧的 Kenney 卡牌、棋子、筹码显得简陋，请不要继续把它们作为主视觉。

请改为：
1. 删除右侧实体棋子和筹码。
2. 删除或极度弱化实体卡牌，避免贴片感。
3. 强化右侧宫殿、云路、日光作为主体。
4. 在宫殿后方或云层中加入低透明淡金阵法圆环。
5. 如需要表达身份推理，只使用两张半透明“牌影轮廓”，不要使用现成卡牌贴图。
6. 牌影可用红/绿微弱光边区分阵营，透明度控制在 10%-20%。
7. 保持左侧标题菜单布局，不要把文字移回中心。

验收标准：
- 第一眼主体是右侧仙山宫殿/云上疑阵，而不是几枚 UI 小图标。
- 右侧没有明显廉价贴片感。
- 画面仍能暗示身份、阵营和推理。
- 不影响开始游戏、游戏测试按钮点击。
```
